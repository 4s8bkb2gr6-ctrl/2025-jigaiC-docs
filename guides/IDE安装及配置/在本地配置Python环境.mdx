## 在本地配置 Python 环境

在开始学习之前，你需要在电脑上配置好 Python 环境。  
我们推荐使用 **VSCode** 或 **PyCharm**，任选其一即可。  

### VSCode

VSCode 是微软推出的轻量级编辑器，结合 **Miniconda** 可以方便地管理 Python 版本和依赖。

### PyCharm

PyCharm 是 JetBrains 开发的专业 Python 集成环境，功能更强大，适合深入学习。  


## Tips

- **VSCode + Miniconda**：适合入门，轻量且易于管理多个 Python 环境。  
- **PyCharm**：功能更强大，适合长期开发。
