#### Mac

1. **安装 Python**
   - 步骤同 VSCode 部分。

2. **安装 PyCharm**
   - 前往 [PyCharm 官网](https://www.jetbrains.com/pycharm/download/) 下载 *Community*（免费社区版）。
   - 安装完成后，打开 PyCharm 并选择一个项目文件夹。

3. **配置运行**
   - 新建 Python 文件 `hello.py`。
   - 输入以下代码：
     ```python
     print("Hello, Python!")
     ```
   - 点击右上角的绿色运行按钮，若输出结果正确，配置完成。
