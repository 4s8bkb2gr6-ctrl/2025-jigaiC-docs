#### Windows

1. **安装 Python**
   - 步骤同 VSCode 部分。

2. **安装 PyCharm**
   - 前往 [PyCharm 官网](https://www.jetbrains.com/pycharm/download/) 下载 Windows 版本（选择 *Community* 免费版）。
   - 安装完成后，打开 PyCharm 并新建项目。

3. **配置运行**
   - 新建文件 `hello.py`，输入：
     ```python
     print("Hello, Python!")
     ```
   - 点击运行，若能正确输出，则配置完成。
