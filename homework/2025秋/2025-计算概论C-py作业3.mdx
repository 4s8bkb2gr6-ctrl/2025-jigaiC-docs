## 02：简单计算器

### 题目描述

一个最简单的计算器，支持+, -, *, / 四种运算。仅需考虑输入输出为整数的情况(除法结果就是商，忽略余数）

**输入**  
输入只有一行，共有三个参数，其中第1、2个参数为整数，第3个参数为操作符（+,-,*,/）。

**输出**   
输出只有一行，一个整数，为运算结果。然而：  
1. 如果出现除数为0的情况，则输出：Divided by zero!   
2. 如果出现无效的操作符(即不为 +, -, *, / 之一），则输出：Invalid operator!   

### 题解
**思路**  
注意输出结尾的感叹号

**知识点**  
<ResponseField name="分支结构" type="第3章程序的控制结构">
</ResponseField>
**代码**  

<Tabs>
  <Tab title="尹思源">
    ```python 02：简单计算器 lines icon="python"
    num1, num2, op = input().split()
    num1 = int(num1)
    num2 = int(num2)
    if op == "+":
        print(num1 + num2)
    elif op == "-":
        print(num1 - num2)
    elif op == "*":
        print(num1 * num2)
    elif op == "/":
        if num2 == 0:
            print("Divided by zero!")
        else:
            print(num1 // num2)
    else:
        print("Invalid operator!")
    ```
  </Tab>

  ```python 02：简单计算器 lines icon="python"
  s = input().split()
  x, y, op = s[0], s[1], s[2]
  if int(y) == 0 and op == "/":
      print("Divided by zero!") 
  elif op not in ["+", "-", "*", "/"]:
      print("Invalid operator!")
  else:
      print(int(eval(x + op + y)))
  ```

## 07：正常血压

### 题目描述

监护室每小时测量一次病人的血压，若收缩压在90 - 140之间并且舒张压在60 - 90之间（包含端点值）则称之为正常，现给出某病人若干次测量的血压值，计算病人保持正常血压的最长小时数。   

**输入**  
第一行为一个正整数n，n < 100
其后有n行，每行2个正整数，分别为一次测量的收缩压和舒张压，中间以一个空格分隔。 

**输出**  
输出仅一行，血压连续正常的最长小时数。

### 题解
**思路**  


**知识点**  
<ResponseField name="循环结构" type="第3章程序的控制结构">
</ResponseField>
**代码**  

<Tabs>
  <Tab title="尹思源">
    ```python 07：正常血压 lines icon="python"
    n = int(input())
    current = 0
    longest = 0
    for _ in range(n):
        a, b = map(int, input().split())
        if 90 <= a <= 140 and 60 <= b <= 90:
            current += 1
            longest = max(longest, current)
        else:
            current = 0
    print(longest)
    ```

## 08：找出全部子串位置

### 题目描述

输入两个串s1,s2，找出s2在s1中所有出现的位置

两个子串的出现不能重叠。例如'aa'在 aaaa 里出现的位置只有0,2

**输入**  
第一行是整数n
接下来有n行，每行两个不带空格的字符串s1,s2

**输出**  
对每行，从小到大输出s2在s1中所有的出现位置。位置从0开始算
如果s2没出现过，输出 "no"
行末多输出空格没关系

### 题解
**思路**  


**知识点**  
<ResponseField name="循环结构" type="第3章程序的控制结构">
</ResponseField>
**代码**  

<Tabs>
  <Tab title="尹思源">
    ```python 08：找出全部子串位置 lines icon="python"
    n = int(input())
    for _ in range(n):
        s1, s2 = input().split()
        positions = []
        i = 0
        while i <= len(s1) - len(s2):
            if s1[i : i + len(s2)] == s2:
                positions.append(i)
                i += len(s2)  # 不允许重叠
            else:
                i += 1
        if positions:
            print(*positions)  # print(*positions) 会自动以空格分隔输出。
        else:
            print("no")
    ```
  </Tab>
   <Tab title="钱丽艳">

**输出**  
由n行数字组成的正实心三角形

### 题解
**思路**  


**知识点**  
<ResponseField name="循环结构" type="第3章程序的控制结构">
</ResponseField>
**代码**  

<Tabs>
  <Tab title="尹思源">
    ```python 09：打印数字三角形 lines icon="python"
    n = int(input())
    for i in range(1, n + 1):
        space = (n - i) * " "
        num = [str(j) for j in range(1, i * 2)]
        print(space + "".join(num))
    ```

    ```python 09：打印数字三角形 lines icon="python"

    for i in range(1, n+1):
        for j in range(1, n-i+1):  # 打空格。 空格的个数：(一行总字符数-一行数字字符数)/2=((2*n-1)-(2*i-1))/2=n-i

            print(j, end="")
    ```
