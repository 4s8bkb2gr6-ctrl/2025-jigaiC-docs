## 03：求三个数的和

### 题目描述

输入三个整数或小数，输出它们的和

如果结果是整数，就保留小数点后面一位的0

请注意：本题应该用float对三个数进行转换

**输入**  
​输入三个整数或小数


**输出**  
和

### 题解
**思路**  


**知识点**  
<ResponseField name="数字类型-浮点型" type="第2章基本数据类型">
</ResponseField>
**代码**  

<Tabs>
  <Tab title="徐志衡">
    ```python 03：求三个数的和 Solution 1 lines icon="python"
    # Solution 1
    # 使用 map 函数将输入的字符串转换为浮点数 float类型
    x, y, z = map(float, input().split())
    a = x + y + z
    # 强烈推荐使用 is_integer() 方法判断浮点数是否为整数
    if a.is_integer():
        print(f'{a:.1f}')
    else:
        print(a)
    ```
    ```python 03：求三个数的和 Solution 2 lines icon="python"
    # Solution 2
    # 使用 for 循环将输入的若干个数字存储在列表中
    l = [float(x) for x in input().split()]
    # 使用 sum 函数计算列表中所有元素的和
    a = sum(l)
    # print()函数会自动将是整数的浮点数保留一位小数，因此对于本题不需要特判结果是否为整数
    print(a)
    ```

## 07：字母加密

### 题目描述

英文小写字母构成一个封闭的环，输入密钥n，在环上按顺时针方向移动n步，得到加密后的字母。注意，密钥的取值范围不仅限于0~26。如输入英文小写字母a，密钥3，则加密后的字母为d；输入英文小写字母z，密钥1，则加密后的字母为a。输入英文小写字母a，密钥27，则加密后的字母为b。

**输入**  
两行  
第一行是英文小写字母  
第二行是密钥

**输出**  
加密后的字母

### 题解
**思路**  


**知识点**  
<ResponseField name="数字类型-浮点型" type="第2章基本数据类型">
</ResponseField>
**代码**  

<Tabs>
  <Tab title="徐志衡">
    ```python 07：字母加密 lines icon="python"
    c = input()
    n = int(input())
    # 字符 a-z 在 Unicode 中的编码是按顺序且连续排列的
    # 先计算输入字符相对于字符 'a' 的偏移量
    offset = ord(c) - ord('a')
    # 再计算输入字符移动 n 步后的偏移量，因为是循环移动，所以需要取余数 26
    newOffset = (offset + n) % 26
    # 最后将偏移量加上字符 'a' 的编码，得到加密后的字符
    print(chr(ord('a') + newOffset))
    ```

</Tabs>
