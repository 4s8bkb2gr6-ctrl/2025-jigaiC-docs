## 在本地配置 Python 环境

在开始学习之前，你需要在电脑上配置好 Python 环境。  
我们推荐使用 **VSCode** 或 **PyCharm**，任选其一即可。  

---

### VSCode

VSCode 是微软推出的轻量级编辑器，结合 **Miniconda** 可以方便地管理 Python 版本和依赖。

#### Mac

1. **使用命令行安装 Miniconda**
   -打开终端：使用`command`+`空格`打开聚焦搜索，输入`terminal`，回车打开终端。
   -在终端中输入以下命令：
   ```bash
    mkdir -p ~/miniconda3
    curl https://repo.anaconda.com/miniconda/Miniconda3-latest-MacOSX-arm64.sh -o ~/miniconda3/miniconda.sh
    bash ~/miniconda3/miniconda.sh -b -u -p ~/miniconda3
    rm -rf ~/miniconda3/miniconda.sh
    ```
    -安装完成后，输入以下命令对 bash 和 zsh shell 进行初始化：
    ```bash
    ~/miniconda3/bin/conda init bash
    ~/miniconda3/bin/conda init zsh
    ```
    -最后在终端输入以下命令，若显示版本号则表示安装成功：
    ```bash
    conda -V
    ```


2. **创建 Python 环境**
   - 在终端中输入：
     ```bash
     conda create -n pycourse python=3.11
     conda activate pycourse
     ```
     这样我们就新建了一个名为 `pycourse` 的 Python 3.11 环境。

3. **安装 VSCode**
   - 前往 [VSCode 官网](https://code.visualstudio.com/) 下载并安装。
   - 打开 VSCode，在扩展（Extensions）中搜索并安装 **Python** 插件。

4. **配置 VSCode 使用 Conda 环境**
   - 打开 VSCode，进入 `hello.py` 文件。
   - 按下快捷键 **Ctrl+Shift+P**（Mac 上为 **Cmd+Shift+P**），输入并选择 **Python: Select Interpreter**。
   - 选择 `pycourse` 环境（通常路径里会带有 miniconda3/envs/pycourse）。
   - 新建文件 `hello.py`，输入：
     ```python
     print("Hello, Python from Conda!")
     ```
   - 点击运行，若能正确输出，说明配置成功。

#### Windows

1. **安装 Miniconda**
   - 打开 [Miniconda 官网](https://docs.conda.io/en/latest/miniconda.html)，下载 Windows 64-bit 安装包。
   - **安装时一定要注意：**
     - 勾选 *Add Miniconda3 to PATH environment variable*（如果有提示）。
     - 或者选择 *Register Miniconda as the system Python 3.11*。
   - 安装完成后，打开 **命令提示符 (CMD)** 或 **Anaconda Prompt**，输入：
     ```bash
     conda --version
     ```
     如果能看到版本号，说明安装成功。

2. **检查系统路径 (PATH)**
   - 如果 CMD 里输入 `conda` 提示“不是内部或外部命令”，说明 PATH 没有配置好。
   - 解决方法：
     1. 打开 **开始菜单** → 搜索 **环境变量** → 点击 **编辑系统环境变量**。
     2. 在 **系统变量** 中找到 **Path** → 点击 **编辑**。
     3. 添加以下路径（假设你安装在 `C:\Users\<你的用户名>\Miniconda3`）：
        ```
        C:\Users\<你的用户名>\Miniconda3
        C:\Users\<你的用户名>\Miniconda3\Scripts
        C:\Users\<你的用户名>\Miniconda3\Library\bin
        ```
     4. 点击 **确定** 保存，重新打开 CMD 测试：
        ```bash
        conda --version
        ```

3. **创建 Python 环境**
   - 在 CMD 或 Anaconda Prompt 输入：
     ```bash
     conda create -n pycourse python=3.11
     conda activate pycourse
     ```

4. **安装 VSCode**
   - 前往 [VSCode 官网](https://code.visualstudio.com/) 下载 Windows 版本并安装。
   - 在 VSCode 扩展市场搜索并安装 **Python** 插件。

5. **配置 VSCode 使用 Conda 环境**
   - 打开一个项目文件夹。
   - 新建文件 `hello.py`，写入：
     ```python
     print("Hello, Python from Conda!")
     ```
   - 按下 **Ctrl+Shift+P** → 搜索并选择 **Python: Select Interpreter**。
   - 选择 `pycourse` 环境。
   - 点击运行，若能正确输出，说明环境配置完成。

### PyCharm

PyCharm 是 JetBrains 开发的专业 Python 集成环境，功能更强大，适合深入学习。  

#### Mac

1. **安装 Python**
   - 步骤同 VSCode 部分。

2. **安装 PyCharm**
   - 前往 [PyCharm 官网](https://www.jetbrains.com/pycharm/download/) 下载 *Community*（免费社区版）。
   - 安装完成后，打开 PyCharm 并选择一个项目文件夹。

3. **配置运行**
   - 新建 Python 文件 `hello.py`。
   - 输入以下代码：
     ```python
     print("Hello, Python!")
     ```
   - 点击右上角的绿色运行按钮，若输出结果正确，配置完成。

---

#### Windows

1. **安装 Python**
   - 步骤同 VSCode 部分。

2. **安装 PyCharm**
   - 前往 [PyCharm 官网](https://www.jetbrains.com/pycharm/download/) 下载 Windows 版本（选择 *Community* 免费版）。
   - 安装完成后，打开 PyCharm 并新建项目。

3. **配置运行**
   - 新建文件 `hello.py`，输入：
     ```python
     print("Hello, Python!")
     ```
   - 点击运行，若能正确输出，则配置完成。

## 总结

- **VSCode + Miniconda**：适合入门，轻量且易于管理多个 Python 环境。  
- **PyCharm**：功能更强大，适合长期开发。

## 常见问题 FAQ

### 1. Miniconda 安装后 CMD 无法识别 `conda` 命令怎么办？
- 说明 PATH 没有配置好，请参考 Windows 部分的 **检查系统路径 (PATH)** 步骤。

### 2. VSCode 里运行时找不到解释器怎么办？
- 按 **Ctrl+Shift+P** → 输入 **Python: Select Interpreter** → 选择 `pycourse` 环境。

### 3. Mac 上为什么用 `python3` 而不是 `python`？
- 因为 Mac 系统自带一个老版本的 `python`（通常是 2.x），为了避免冲突，Miniconda 默认用 `python3`。

### 4. 我可以不用 Miniconda，直接装 Python 吗？
- 可以，但推荐 Miniconda，因为它能方便地管理环境，不会弄乱系统自带的 Python。

- 在 VSCode 新建 `hello.py` 文件，写入：
